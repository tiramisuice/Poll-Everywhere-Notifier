Icon Placeholder Files

This extension requires three icon sizes:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

You can create these icons using any image editor or online icon generator.

Suggested design:
- A bell icon (for notifications)
- Poll Everywhere blue color (#1AB2D7)
- Simple, clean design

Quick ways to create icons:
1. Use an online icon generator like https://www.favicon-generator.org/
2. Use Canva or similar design tools
3. Use an AI image generator
4. Export from design software like Figma or Adobe Illustrator

Once created, place the three PNG files in this icons/ directory.
